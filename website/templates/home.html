Welcome to the home page of David's Habit Tracker!

Quick introduction:

There are four tabs in total that can be accessed using the buttons above or the navbar in the top.

The website itself has two main functions, with the Task Tracker being the main one:



  Task Tracker

Track daily, weekly or monthly tasks.

Earn XP to level up & claim rewards for each Level-Up

   

  Notes

Add Notes or Reminders to a list

Sorted by Date added




The other functions are related to the Task Tracker functionality:




  Reward Center

Add custom rewards & claim them at every Level-Up.

 

You can also see the Reward History, with each Reward being listed along with the date it was added

   

  Profile

Displays various information about the User

This includes current and total XP, the number of completed tasks and the current level.

You can also set custom XP rewards for all task types and a custom XP requirement for each level there
